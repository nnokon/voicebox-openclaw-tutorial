# OpenClaw + Voicebox Local TTS Configuration Tutorial

🇨🇳 [中文版本](#-中文版本) | 🇬🇧 **English**

Complete guide for configuring OpenClaw with local AI voice (TTS) functionality, with voice cloning support.

## 📚 Content

- **Complete Tutorial** - Detailed step-by-step instructions from scratch
- **Voice Cloning Guide** - How to extract audio samples from Fish Audio
- **Troubleshooting** - Solutions for common problems
- **Auto-startup** - Automatically launch Voicebox and proxy service

## 🚀 Quick Start

1. Download [Voicebox](https://github.com/jamiepine/voicebox)
2. Download Qwen TTS model from [Hugging Face](https://huggingface.co/Qwen/Qwen3-TTS-12Hz-1.7B-Base/tree/main)
3. Follow the steps in `TUTORIAL_EN.md`
